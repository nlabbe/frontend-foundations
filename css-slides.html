What's CSS?

Cascading Style Sheets (CSS) is a simple mechanism for adding style (e.g., fonts, colors, spacing) to Web documents.

W3C

  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files

Which benefits will CSS give me?

  • Control layout of many documents from one single style sheet
  • More precise control of layout
  • Apply different layout to different media-types (screen, print, etc.)
  • Numerous advanced and sophisticated techniques.

How does CSS work?

The basic CSS syntax

A CSS rule-set consists of a selector and a declaration block:

  • The selector points to the HTML element you want to style.
  • The declaration block contains one or more declarations separated by semicolons.
  • Each declaration includes a CSS property name and a value, separated by a colon.
  • A CSS declaration always ends with a semicolon, and declaration blocks are surrounded by curly braces.

Example

Using HTML we could have done it like this:


							<body bgcolor="#FF0000"></body>

With CSS the same result can be achieved like this:


							body {background-color: #FF0000;}

CSS Selectors

Element Selector

The element selector selects elements based on the element name.


							p {
							    text-align: center;
							    color: red;
							}

id Selector

The id selector uses the id attribute of an HTML element to select a specific element.


							#foo {
							    text-align: center;
							    color: red;
							}

Class Selector

The class selector selects elements with a specific class attribute.


							.center {
							    text-align: center;
							    color: red;
							}

You can also specify that only specific HTML elements should be affected by a class.


							p.center {
							    text-align: center;
							    color: red;
							}

HTML elements can also refer to more than one class.


			
This paragraph refers to two classes.

Grouping Selectors


					h1, h2, p {
							    text-align: center;
							    color: red;
							}

Applying CSS to an HTML document

Method 1: In-line (the attribute style)


								<html>
								  <head>
									<title>Bazzinga</title>
								  </head>
								  <body style="background-color: #FF0000;">
									<p>Lorem ipsum.</p>
								  </body>
								</html>

Method 2: Internal (the tag style)


								<html>
								  <head>
									<title>Bazzinga</title>
									<style type="text/css">
									  body {background-color: #FF0000;}
									</style>
								  </head>
								  <body>
									<p>Lorem ipsum.</p>
								  </body>
								</html>

Method 3: External (link to a style sheet)


								<!DOCTYPE html>
								<html>
								<head>
									<title>Bazzinga</title>
									<link rel="stylesheet" type="text/css" href="style/style.css" />
								</head>
								<body>

								</body>
								</html>
